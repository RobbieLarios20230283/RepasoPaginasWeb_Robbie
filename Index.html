¿Qué es HTML y cuál es su función principal en el desarrollo web?

HTMLes el lenguaje estándar para la creación y estructuración de páginas web.

¿Qué son las etiquetas HTML y cómo se utilizan?

Las etiquetas son elementos que indican cómo se debe estructurar el contenido en una página web.

Ejemplos de algunas de las etiquetas HTML más comunes

    h1: Encabezados principales.
    p: Párrafos de texto.
    a: Enlaces a otras páginas.
    img : Imágenes.
    div : Contenedores para estructurar contenido.

¿Qué es CSS y cómo se utiliza para dar estilo a una página web?

CSS es un lenguaje se utiliza para editar y formar una pagina web de un documento HTML que permite controlar el diseño de la pagina

¿Cuáles son las propiedades más importantes de CSS?

    color: Define el color del texto.
    background color: Establece el color de fondo.
    font size: Especifica el tamaño de la fuente.
    margin: Controla los márgenes externos de los elementos.
    padding: Establece el espacio interno de los elementos.

Explica el modelo de caja (box model) en CSS

El modelo de caja en CSS es cómo los elementos HTML son estructurados en en distintos formatos.

¿Qué es el diseño responsivo y cómo se logra con CSS?

El diseño responsivo es una técnica que asegura que las páginas web se vean bien en diferentes dispositivos, como teléfonos móviles, tabletas y computadoras de escritorio.

¿Qué es JavaScript y cómo se integra en el desarrollo web?

JavaScript es un lenguaje de programación que se utiliza para agregar interactividad y dinamismo a las páginas web. Se integra con HTML para manipular el contenido y con CSS para modificar los estilos de manera dinámica.

Explica cómo JavaScript interactúa con el HTML y CSS a través del DOM (Document Object Model)

El DOM es una representación estructural de la página web en la memoria. JavaScript interactúa con el DOM para modificar el contenido HTML y los estilos CSS de manera dinámica. A través del DOM, JavaScript puede añadir, eliminar o modificar elementos HTML y aplicar cambios en el estilo CSS.

Ejemplos de cómo JavaScript puede hacer que una página web sea interactiva

    Mostrar y ocultar elementos al hacer clic.
    Validar formularios antes de enviarlos.
    Actualizar contenido de manera dinámica sin recargar la página .

¿Qué es Git y por qué es importante para el desarrollo web?

Git es un sistema de control de versiones que permite a los programadores lleven un registro de los cambios realizados en el código fuente a lo largo del tiempo. Es importante porque facilita la trabajo entre desarrolladores y asegura que los cambios sean visualizables.

Explica cómo Git se usa para el control de versiones en proyectos de desarrollo web

Git permite a los desarrolladores gestionar el código fuente de un proyecto web mediante el uso de repositorios.

¿Qué es GitHub y cómo facilita la colaboración en proyectos web?

GitHub es una plataforma en línea que utiliza Git para alojar repositorios de código .
